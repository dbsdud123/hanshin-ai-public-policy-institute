# 프로젝트 개요

## 프로젝트명

한신대학교 AI 공공정책연구소 공식 웹사이트 리뉴얼

## 현재 상태

교수님이 만든 원본은 Next.js 14 기반의 연구소 웹사이트입니다. 공개 배포 주소는 `https://ai-policy-institute.vercel.app/`이며, GitHub 공개 저장소 `LeeSeogMin/AI_center`의 `master` 브랜치 커밋 `59b062b33e0dea9194f095b41e2c969603de4cf9`와 아이클라우드 원본 파일 구성이 일치합니다.

현재 원본은 홈페이지, 소개, 연구진, 연구성과, 게시판, 로그인/회원가입, 관리자 페이지를 포함합니다. 다만 관리자가 코딩 없이 핵심 페이지를 수정하는 CMS 기능은 아직 완성되어 있지 않습니다.

## 리뉴얼 핵심 방향

- 공식 연구소 웹사이트다운 신뢰감 있는 정보 구조와 디자인
- 모바일/데스크톱 모두에서 완성도 있는 반응형 UI
- 연구소 소개, 연구실, 연구진, 자문위원, 연구성과, 공지사항, 문의 정보를 관리자 화면에서 수정
- Supabase 기반 인증, DB, 파일 업로드 구조 정비
- 운영자가 실수해도 쉽게 복구할 수 있는 콘텐츠 관리 정책

## 중요한 결정

기술 기반은 원본과 같은 Next.js + Supabase를 유지하는 것이 좋습니다. 이유는 기존 코드에 이미 인증, 게시판, Supabase 연동 패턴이 있고, Vercel 배포와도 잘 맞기 때문입니다.

다만 그대로 이어 쓰기보다 다음을 재설계해야 합니다.

- 소개/연구진/연구성과 하드코딩 제거
- 관리자 페이지를 단순 목록이 아닌 에디터형 CMS로 확장
- DB 스키마를 실제 운영 콘텐츠 중심으로 다시 설계
- 공개 개인정보 노출 범위 재검토
- 테스트용 샘플 데이터 제거
